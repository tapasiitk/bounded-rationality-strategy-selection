## Overview

This repository contains the complete computational analysis and reproducible research code for **"When Bounded Rationality Predicts Optimal Behavior"** ‚Äî a study investigating when humans behave rationally versus employing decision shortcuts.

**The Core Question:** Are systematic biases in human choice evidence of cognitive limitations, or adaptive responses to task demands?

**The Answer:** When information is easy to integrate (commensurable attributes), humans predominantly employ optimal strategies (97.4% of the time), eliminating systematic biases. This suggests that **bounded rationality can predict optimal behavior** when environmental conditions support it.

## Theoretical Contribution

Rather than viewing decision biases as universal human limitations, this work demonstrates that:

1. **Strategy selection is context-dependent** ‚Äî Individuals probabilistically choose between optimization and satisficing based on perceived cognitive effort.
2. **Commensurability is a boundary condition** ‚Äî When attributes can be easily integrated, optimal processing emerges naturally.
3. **Bounded rationality predicts rationality** ‚Äî Simon's framework can specify when humans will behave optimally, not just when they fail.

## Computational Methods

### Models Implemented

**1. Effort-Based Strategy Selection Model** (Proposed)
- Individuals probabilistically choose between optimization and satisficing
- Optimization: Sum all information attributes
- Satisficing: Attribute-wise pairwise comparisons
- Parameter: Œª (probability of optimization)

**2. Rank-Based Salience Model** (Baseline)
- Weights values based on momentary rank within each frame
- Explains systematic biases through salience-driven processing
- Parameters: w‚ÇÅ, w‚ÇÇ, w‚ÇÉ (rank weights)

### Validation Framework

‚úÖ **Maximum Likelihood Estimation** with parameter bounds  
‚úÖ **5-Fold Cross-Validation** to assess generalization  
‚úÖ **Parameter Recovery** testing to verify model identifiability  
‚úÖ **Robustness Analysis** across multiple random initializations  
‚úÖ **Bayesian Hypothesis Testing** (BF‚ÇÄ‚ÇÅ = 21.67 for null attraction effect)

## Design Considerations

- **Commensurable Attributes**: All values presented as numerical magnitudes that can be directly compared
- **Multi-Attribute RSVP Paradigm**: 24 sequential frames (500ms each) before choice
- **Trials: 72 main trials + 24 catch trials**
- **Sample**: N = 17 participants (after exclusion criteria)
- **Trials**: 1,506 valid trials (92.3% retention after quality control)
